# Conjugate Residual Acceleration of Proximal Gradient Methods

## Description
MATLAB codes for the paper "Conjugate residual acceleration of proximal gradient methods with optimal trial vectors".

## Requirements
- MATLAB 
- Optimization toolbox required

## How to run
Run the following script:
    Run_l1_logistic.m
## Data
Synthetic data is generated within the script.

mxn=6000x3000, cond_num=1e3, 1e5.

mxn=2000x5000, cond_num=1e3, 1e8.
